# Trustless NFT Purchase Contract
Allow two parties to transfer an NFT for a specified amount.

## Contract Owner
- Set Buyer
- Set Seller

## Seller
- Set Payment Amount
- Transfer NFT

## Buyer
- Set NFT
- Set Payment
- Withdraw when Seller fails to transfer

## Test
Tests are written in `tests/test-trustless-nft-purchase.js`
```shell
npm run test
```
